Task: Implement arm_neon.h header for ARM NEON intrinsic type support
Status: in_progress

Goal: Create a bundled arm_neon.h header with NEON types (uint8x16_t, int8x16_t, etc.)
and intrinsic functions (vld1q_u8, vst1q_u8, veorq_u8, etc.) needed for Linux kernel
RAID6 NEON code and crypto AEGIS128 NEON code.

Approach: Follow the same pattern as x86 SSE headers (emmintrin.h etc.) using
struct-based types and inline functions with inline assembly.

Also enable __ARM_NEON macro for aarch64 targets.
